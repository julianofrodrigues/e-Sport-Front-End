## 📁 Project Access
To try it is very simple:<br>
-Clone the repository
```bash
$ git clone https://github.com/julianofrodrigues/e-Sport-Front-End
```
-Download the libraries
```bash
$ yarn install
```
Or
```bash
$ npm install
```
-Start the Project
```bash
$ yarn dev
```
Or
```bash
$ npm dev
```
